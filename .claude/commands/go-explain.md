# Go Code Explain

## 対象

$ARGUMENTS

## 解説手順

1. 対象コードを読み込む
2. Go の設計思想に基づいて解説
3. 以下の観点で説明

## 解説の観点

### 1. 概要
- このコードは何をしているか
- なぜこのように実装されているか（Go の思想との関連）

### 2. 詳細解説
- 各部分の役割と動作
- 使用されているパターンやイディオム
- 重要な Go の機能（interface, goroutine, channel など）

### 3. 設計のポイント
- なぜこの設計が選ばれたか
- 代替案との比較
- トレードオフ

### 4. 学習ポイント
- このコードから学べる Go のベストプラクティス
- 関連する Go Proverbs
- 参考にすべきスキル
